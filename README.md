# SchoolFlow

MVC-приложение для школьных задач на ASP.NET Core 8 и SQLite.

Полная документация проекта: [DOCUMENTATION.md](DOCUMENTATION.md)

## Запуск через Docker

```powershell
docker compose up --build
```

После запуска откройте:

```text
http://localhost:8080
```

Данные SQLite хранятся в Docker volume `schoolflow-data`, поэтому задачи сохраняются между перезапусками контейнера.

## Кабинет учителя

Демо-доступ создается автоматически при первом запуске базы:

```text
Логин: teacher
Пароль: teacher123
```

После входа доступны разделы `Учитель`, `Класс`, импорт Excel и выдача задач конкретным ученикам.

## Кабинет ученика

Для чистой демо-базы автоматически создается ученик:

```text
Логин: student
Пароль: student123
```

Ученик видит только задачи, назначенные именно ему, и может отметить задачу выполненной.

## Импорт класса из Excel

Файл должен быть в формате `.xlsx`. Первая строка может быть заголовком.

Ожидаемые колонки:

```text
ФИО | Класс | Email | Логин | Пароль
```

Email, логин и пароль необязательны. Если логин не указан, приложение создаст его автоматически. Если пароль не указан, используется `student123`. Дубли по сочетанию `ФИО + Класс` не импортируются повторно.

## Полезные команды

Остановить приложение:

```powershell
docker compose down
```

Остановить и удалить сохраненную базу:

```powershell
docker compose down -v
```

Запустить локально без Docker:

```powershell
dotnet run --project .\swa.csproj
```
